Bandwidth Hero Proxy

Welcome to the Bandwidth Hero Proxy!

To use the proxy in the Bandwidth Hero extension, follow these steps:

  1. Open the Bandwidth Hero extension settings.
  2. In the Data Compression Service section, add the following URL: https://your-netlify-domain.netlify.app/api/index.
  3. Save the settings and you are good to go!

For more information and the source code, visit the GitHub repository.
